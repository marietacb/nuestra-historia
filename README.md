
# Love Analytics ❤️

¡Esta es la aplicación personalizada para nuestra historia!

## Cómo verla en tu portátil (Desarrollo)

1. **Descarga todos los archivos** en una carpeta.
2. Asegúrate de tener instalado **Node.js** ([descárgalo aquí](https://nodejs.org/)).
3. Abrir una terminal (consola) en esa carpeta.
4. Ejecutar los siguientes comandos:
   ```bash
   npm install
   npm run dev
   ```
5. Abre el enlace que aparecerá (normalmente `http://localhost:5173`).

## Cómo compartirla con tu pareja (Despliegue)

Para que Guille pueda entrar desde su propio portátil en su casa:

1. Crea una cuenta gratuita en [Vercel](https://vercel.com/) o [Netlify](https://www.netlify.com/).
2. Arrastra la carpeta del proyecto a su panel de control.
3. Te darán un enlace público (ej. `nuestra-historia.vercel.app`).
4. **¡Paso Importante para sincronizar!**: Como los datos se guardan en el navegador por privacidad, ella verá la app vacía al principio. 
   - Ve a **Ajustes** en TU ordenador.
   - Pulsa "Descargar Nuestra Historia".
   - Envíale ese archivo `.json` por chat.
   - El debe entrar en el enlace desde SU portátil, ir a **Ajustes** y pulsar "Cargar Historia Compartida".

¡Disfrutad de vuestros recuerdos!
